# One-Line
One Line，一行一句话，简单粗暴的知识类app，尽可能的高度抽象解释对应的索引，1.5MB大小，可根据需求自定义条目的类百科app，在线查询功能和离线手册。

<h3>目前只有Android版</h3>
谷歌与维基百科版，已有梯子，适合学术追求人群：http://cohbird.space/oneline.apk

百毒与百毒百科版，没有梯子，适合生活科普人群：http://cohbird.space/oneline_baidu.apk
